# 要件定義

## 1. プロジェクト背景

- サイト名: 埼玉きのこ研究会
- URL: https://www.ippon.sakura.ne.jp/
- 開設: 2000年8月15日（26年目）
- ホスティング: さくらインターネット共有ホスティング
- 現行技術: HTML 4.01 / Shift_JIS / TABLE レイアウト / Homepage Builder

### リニューアルの動機

- SSL証明書エラーによるブラウザ警告
- モバイル非対応（固定幅752px）
- 更新作業の非効率（Homepage Builder + FTP）
- 技術的負債の蓄積（25年間の運用）

## 2. 現行サイトの規模

| 項目 | 数量 |
|------|------|
| 総HTMLページ数 | 460〜680ページ |
| 総リソース数（画像含む） | 2,500〜3,500ファイル |
| 総サイズ | 632MB |
| 例会報告 | 25年分（2000〜2025）、460ページ、2,800画像 |
| 会報バックナンバー | 35号分（1993〜2024）、96ページ |
| 外部リンク | 16件中6件死亡 |

## 3. 機能要件

### 3.1 必須ページ

| ページ | 内容 | 更新頻度 |
|--------|------|----------|
| トップページ | お知らせ、ナビゲーション | 月1〜2回 |
| 行事予定 | 年間スケジュール | 年1回（4月） |
| 例会報告 | 観察会写真+テキスト | 年8回 |
| 例会報告アーカイブ | 2000〜現在の全報告 | 移行時のみ |
| 会報バックナンバー | 第7号〜最新号 | 年1回 |
| 入会案内 | 入会方法・会費 | 変更時 |
| 会について | 沿革・会則・会員数 | 変更時 |
| きのこ関連データ | 学名辞書・読み方 | ほぼ静的 |
| リンク集 | 関連団体・個人サイト | 変更時 |

### 3.2 新規追加要件

| 機能 | 優先度 |
|------|--------|
| レスポンシブデザイン（モバイル対応） | 必須 |
| SSL/HTTPS対応 | 必須 |
| SEO対応（meta description, OGP, sitemap.xml） | 必須 |
| 写真ギャラリー表示 | 必須 |
| 問い合わせフォーム（個人情報直接掲載の代替） | 推奨 |
| アクセス解析 | 推奨 |
| ダークモード | 任意 |

### 3.3 非機能要件

| 項目 | 要件 |
|------|------|
| 表示速度 | Lighthouse Performance 90点以上 |
| アクセシビリティ | Lighthouse Accessibility 90点以上 |
| SEO | Lighthouse SEO 90点以上 |
| 対応ブラウザ | Chrome, Safari, Firefox, Edge（最新2バージョン） |
| 対応端末 | スマートフォン、タブレット、デスクトップ |

## 4. 制約条件

| 制約 | 内容 |
|------|------|
| 更新者 | 塚本歩（技術者、HTML/git/CLI操作可能） |
| 予算 | 独自ドメイン費用（年1,000〜3,000円）のみ |
| 移行期間 | 既存サイトと並行運用が必要 |
| コンテンツ | 25年分のアーカイブを完全に移行 |

## 5. 個人情報に関する対応方針

現行サイトに掲載されている個人情報:

| 情報 | 掲載箇所 | 対応方針 |
|------|----------|----------|
| 事務局住所・電話番号 | 会則ページ | 会に確認の上、問い合わせフォームに置換推奨 |
| 入会担当者住所・電話番号・メール | 入会案内 | 同上 |
| 代表メールアドレス | トップページ | フォームに置換、またはメール表示を画像化 |
| 世話人電話番号 | 行事予定 | 会に確認の上、削除または限定公開 |

## 6. 旧サイトからの移行要件

### 必ず移行

- 例会報告全件（2000〜現在）+ 写真
- 会報バックナンバー全件
- 行事予定（最新年度）
- 入会案内・会則・沿革

### 移行時に整理

- 死亡外部リンク6件の削除
- テレホーダイ記述の削除
- LZHファイルのZIP変換
- フレームセット構造の解体（各地の会報紹介）
- MSOスタイルのクリーンアップ

### 移行不要

- WordPress関連ファイル（wp/ディレクトリ）
- バックアップファイル（.bak, .htm_）
- Homepage Builderゴミ箱（__HPB_Recycled）
- Thumbs.db
